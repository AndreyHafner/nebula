README

This project is a template which defines minimal build.properties, one or more map files, 
and customTargets.xml, if required.

Additional Optional files:
buildExtra.xml (overrides to generic build.xml)

allElements.xml
customTargets.xml
customAssembly.xml
myproduct.product
pack.properties (to disable packing for some plugins)

---------------------------------------------------------------------------

Gotchas:

You must ensure your plugins set BREE correctly, since the build will NOT.
